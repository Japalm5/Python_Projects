# Python_Projects
fun projects to showcase my coding progress

Pantry_Fridge_main is a program which converts user choices into actions that manipulate persistent dictionary files governing inventory count and recipes. 
It features the ability to add and delete recipes, add or remove inventory count at will, "make" a recipe that automatically adjusts ingredient count and 
complete handling of invalid input.

HangManGame is a program replicating the popular game hangman. users are given 7 guesses to guess the correct letters that spell out the chosen word. 
Careful, guessing the same letter again counts as a wrong guess!
